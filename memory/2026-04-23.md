## 18:23 - 修复 render 函数多余 `)` bug，JS 语法验证通过

### 最终问题定位
render 函数中的 JS 代码：
```javascript
el.innerHTML = _ov.replace("{{body}}", _b[_s] ? _b[_s]() : ""));  // 错误：3个)
el.innerHTML = _ov.replace("{{body}}", _b[_s] ? _b[_s]() : ""));   // 正确：2个)
```

三元表达式 `_b[_s] ? _b[_s]() : ""` 本身不需要括号包裹，所以：
- `_ov.replace("{{body}}", _b[_s]?_b[_s]():""))` 意味着 replace 的 `)` + 三元多余的 `)` 
- `_ov.replace("{{body}}", _b[_s]?_b[_s]():""))` 才是正确的 2 个 `)`
