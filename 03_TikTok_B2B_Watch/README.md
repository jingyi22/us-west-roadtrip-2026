## 项目概述
为手表制造工厂设计的 TikTok B2B 营销研究，包含真实 TikTok 账号调研、钩子文案库、内容选题框架。

- `TikTok_B2B_Watch_Research_Report.docx` — 完整研究报告（含调研方法、账号案例、钩子类型）

## 6大钩子类型
1. **成本揭秘型** — 工厂价格 vs 零售价对比
2. **揭秘/曝光型** — "Most luxury watches are assembled in China"
3. **VS对比型** — 工厂 vs 品牌差异
4. **工厂实拍型** — 欢迎来到我们的手表工厂
5. **痛点解决型** — 找不到靠谱供应商？
6. **数据背书型** — 5万只/月，99.5%合格率
