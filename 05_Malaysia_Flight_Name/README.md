# ✈️ 马来西亚航班姓名优化 UX 原型

## 项目概述
针对马来西亚航班乘客姓名填写场景设计的 UX 原型，解决 MRT 码（机读旅行证件代码）输入体验问题。

## 主要文件
- `MY_Name_Optimization_Prototype.html` — 交互式 UX 原型
- `Malaysian_Flight_Name_Optimization.pdf` — 原始 PDF 参考文档

## 核心功能
- 手机端适配（iPhone 风格界面）
- MRT 码智能扫描（MRZ Scanner）
- 航班信息展示
- 乘客信息表单（姓名自动格式化）
- 错误提示与实时校验
- 弹窗确认交互

## 设计特点
- 深色主题 UI
- 渐进式表单填写
- 多种姓名格式示例卡片
- Toast 通知提示
- 模态弹窗交互
